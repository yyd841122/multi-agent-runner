### 核心逻辑

1. 每轮：读取 tasks.md → 找 pending → 标记 in_progress → 生成 prompt → 调用 Claude Code → 保存结果 → 判断结果
2. 成功且有完成证据 → 标记 done → 继续下一轮
3. 成功但缺完成证据 → 停止循环，保持 in_progress
4. 失败 → 停止循环，保持 in_progress
5. 无 pending 任务 → 结束
6. 达到最大轮数 → 结束
