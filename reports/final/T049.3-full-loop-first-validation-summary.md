## 4. Reviewer BLOCKED 原因

Reviewer 调用 DeepSeek API 时，因缺少 `DEEPSEEK_API_KEY` 环境变量而失败。

`run_reviewer_for_game_task` 内部通过 `model_adapter` 调用 DeepSeek，但 `model_adapter` 没有自动加载 `.env` 文件。

这不是业务代码问题，而是环境配置问题。

## 5. .env 安全处理

确认 `.env` 已被 `.gitignore` 忽略：

