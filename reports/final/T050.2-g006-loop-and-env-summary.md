## 4. Reviewer 阶段环境问题

首次 full loop 时 Reviewer 被阻塞，原因：

- `DEEPSEEK_API_KEY` 未加载到环境变量
- `model_adapter` 调用 DeepSeek API 时缺少 Key
- 这是环境配置问题，不是业务代码问题

解决方案：

- 用户手动创建 `.env` 文件并写入 `DEEPSEEK_API_KEY`
- 确认 `.env` 已被 `.gitignore` 忽略
- 手动补跑 `review-game-task G006` 和 `decide-game-task G006`
- 后续 T050.1 添加了 `.env` 自动加载能力，不再需要手动加载

## 5. .env 自动加载能力

T050.1 完成的内容：

- 新增 `tools/env_loader.py`（纯 Python，无第三方依赖）
- `runner.py` 启动时自动加载 `.env`
- 支持 `KEY=VALUE`、`KEY="VALUE"`、`KEY='VALUE'`、`# 注释`
- 默认不覆盖已有系统环境变量
- 新增 `.env.example`（安全示例文件，可提交）
- `.env` 已被 `.gitignore` 忽略
