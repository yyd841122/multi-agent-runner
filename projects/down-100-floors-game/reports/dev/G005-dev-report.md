## 完成内容

1. **平台 HTML 元素**：在 `game-area` 内添加了 5 个 `<div class="platform">` 元素，使用 `data-index` 标识。
2. **平台 CSS 样式**：平台使用绝对定位（`position: absolute`），高度 12px，背景色 `#2a9d8f`（青绿色），带 `#264653` 深色边框和 4px 圆角，`z-index: 5`（低于玩家的 10），初始 `display: none`。
3. **平台固定布局**：5 个平台分布在 y=120、200、280、360、440 的位置，宽度占游戏区域 45%-55%，左右错落排列。
4. **平台显示逻辑**：点击"开始游戏"时调用 `initPlatforms()` 显示所有平台，调用 `resetUI()` 时调用 `hidePlatforms()` 隐藏所有平台。
